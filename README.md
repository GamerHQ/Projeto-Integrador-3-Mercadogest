# Projeto-Integrador-3-Mercadogest
Desenvolver um aplicativo web de arquitetura híbrida (Tauri + Rust + PostgreSQL) voltado à automação de caixa e gestão financeira integrada de um pequeno mercado onde: Agilizar a frente de caixa (PDV). Auxiliará no controle do estoque. Auxiliará no controle financeiro. Ser um sistema com boa performance e pouco uso de memória.
